🚀 Sorting Visualizer

An interactive web-based application that visually demonstrates how different sorting algorithms work in real-time. This project helps in understanding algorithm behavior, time complexity, and optimization through animations.

📌 Features
🎯 Visual representation of sorting algorithms
⚡ Adjustable speed control
📊 Dynamic array size selection
🔄 Generate new random arrays
🧠 Displays Time & Space Complexity
📚 Multiple algorithms included:
Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
Heap Sort
🛠️ Tech Stack
HTML – Structure
CSS – Styling & UI
JavaScript – Logic & animations
